# MQTT Integration Entities
On this page, you can view all the entities related to MQTT integration.

## Buttons

### Save MQTT Configuration <Badge type="tip" text="mqtt_save_settings" />
Saves the MQTT credentials and reconnect.


## Numbers

### MQTT Broker Port <Badge type="tip" text="mqtt_broker_port" /> <Badge type="info" text="Disabled by default" />
Sets the MQTT Broker Port.


## Texts

### MQTT Broker Address <Badge type="tip" text="mqtt_broker_address" />
Sets the MQTT Broker Address.

### MQTT Broker Username <Badge type="tip" text="mqtt_broker_username" /> <Badge type="info" text="Disabled by default" />
Sets the MQTT Broker Username.

### MQTT Broker Password <Badge type="tip" text="mqtt_broker_password" /> <Badge type="info" text="Disabled by default" />
Sets the MQTT Broker Password.
